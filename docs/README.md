# Trade Documentation

This directory is the canonical home for project documentation.

## Structure

- `architecture/` — core architecture and behavior design
- `systems/` — implemented gameplay systems and operator usage
- `plans/` — forward-looking roadmaps and backlog-style plans
- `trees/` — detailed knowledge trees with prerequisites for skills, materials, tools, structures, and emergent paths

## Canonical Documents

- `architecture/hierarchical-goals.md` — Goal planning and decomposition
- `architecture/occupation-and-economy-framework.md` — Progression pacing, tribe/town/traders identity, cross-triad interactions
- `systems/invention-system.md` — Autonomous discovery mechanics and quality systems
- `plans/memory-system-roadmap.md` — Memory clustering, routes, social sharing implementation
- `plans/civilization-roadmap.md` — Multi-branch civilization progression (military, civic, merchant, educational)
- `plans/tradegamenotes-alignment-plan.md` — Workspace-grounded enhancement plan mapped from early TradeGameNotes concepts
- `trees/` — Knowledge prerequisite chains:
  - `skills.md` — Skill unlocks and synergies
  - `materials.md` — Raw and processed materials
  - `tools.md` — Hand tools and equipment
  - `structures.md` — Buildings and infrastructure
  - `knowledge.md` — Invention and pondering
  - `tribal-military.md` — Combat, hunting, territory path
  - `civic-industrial.md` — Settlement, production, governance path
  - `mercantile-entrepreneurial.md` — Trade networks and merchant enterprise path

## Notes

Legacy one-off session logs and implementation summaries were removed from the repository root and `solo/` in favor of these living documents.
